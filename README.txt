#Correr localmente
Para correr a página localmente basta guardar o código localmente com todos os ficheiros.
A organização da pasta deverá ser a seguinte:
* PaginaFolder
	* css
		* reset.css
		* style.css
	* images
		* ajax-loader.gif
		* bg_menu.png
		* loader.gif
		* logo.png
		* social-btns.png
	* js
		* hello.js
		* scripts.js
		* jquery.localscroll-min.js
		* jquery.scrollTo-1.4.2-min.js
		* jquery.tipsy.js
		* modernizr-1.7.min.js
	* shadowbox
		* shadowbox.css
		* shadowbox.js
	* index.html
	* README.txt
